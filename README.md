# SmartCar 智能小车项目

这是一个基于 8051 架构单片机（<REGX52.H>）的智能循迹小车控制程序。项目实现了六路红外传感器的黑线循迹、十字路口检测、PWM 电机调速，以及在特定时间窗口内停车等功能。

## 功能特性

- **六路红外循迹模块**: 左侧两路 (Left_0, Left_1)、中间两路 (Middle_0, Middle_1)、右侧两路 (Right_0, Right_1) 分布组合，支持精确的赛道识别和转弯响应。
- **十字路口探测**: 可稳定识别黑线十字路口（Left_1, Middle_0, Middle_1, Right_0 同时检测到黑线）。
- **PWM 电机驱动调速**: 使用 Timer0 和 Timer1 产生 PWM 信号分别控制左右电机转速。提供不同等级的前进、左转、右转速度，支持急转弯（差速/反转）。
- **时间窗口停车机制**: 通过定时器计测赛道行驶时间（Period 变量），在 191000 ~ 193000 运行周期内识别到特定十字路口时，小车将自动停车。

## 文件组成

- \main.c\: 含有主循环逻辑，定时器初始化，以及在特定计时周期配合十字路口检测控制车辆停行的核心判断。
- \carTracking.c\ / \carTracking.h\: 循迹逻辑处理核心。负责读取六路传感器引脚的高地电平，对赛道情况分类产生控制指令；并提供了 \CrossroadDetection()\ 函数进行路口计数及标志更新。
- \carMovement.c\ / \carMovement.h\: 底层电机控制模块。完成 Timer0 与 Timer1 中断服务的定义产生 PWM 波，对外提供前后左右的高级动作封装（如 \carMoveForward\, \carTurnLeft\, \carStop\ 等）。

## 编译环境

- **IDE**: Keil uVision (包含在 main.uvproj 等工程文件中) 
- **目标芯片**: STC89C52 (或同系列兼容 8051 的单片机)

## 引脚分配说明

**驱动模块接线 (L298N或其他):**
- 左电机驱动引脚：IN1F = P2^0, IN2F = P2^1, IN1B = P2^4, IN2B = P2^5
- 右电机驱动引脚：IN3F = P2^2, IN4F = P2^3, IN3B = P2^6, IN4B = P2^7

**六路红外传感器 (黑线为1，白线为0):**
- 左引脚：P0^5, P0^4
- 中引脚：P0^3, P0^2
- 右引脚：P0^1, P0^0

## 运行流程

1. 上电后，延时 300ms。
2. 初始化定时器 0 和定时器 1。
3. 进入主循环：
    - 不断检测十字路口状态：CrossroadDetection()。
    - 判断是否在目标时间区域 Period > 191000 且 Period < 193000 并检测到路口状态发生变化，若符合则调用 carStop()。
    - 否则调用 Tracking() 处理六路传感器信号控制 PWM 从而实现实时动态转弯寻线。
